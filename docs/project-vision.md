# IAM Platform Lab

## Objective

Build a modern IAM platform from scratch to learn and demonstrate IAM, DevOps and Cloud engineering skills.

## Features

### Identity Management
- Create users
- Disable users
- Manage roles
- Manage groups

### Authentication
- OpenID Connect (OIDC)
- OAuth2
- Single Sign-On (SSO)

### Administration Portal
- User management interface
- Role assignment
- Access review

### Audit
- Track administrative actions
- Store audit logs

## Planned Technologies

### IAM
- Keycloak

### Backend
- Python
- FastAPI

### Frontend
- Streamlit

### Database
- PostgreSQL

### DevOps
- GitHub
- GitHub Actions
- Docker
- Kubernetes

### Cloud
- Terraform
- Azure

## Learning Goals

- Learn Docker and containerization
- Learn Kubernetes orchestration
- Learn Infrastructure as Code with Terraform
- Learn OIDC and OAuth2
- Learn CI/CD pipelines
- Build a portfolio project for IAM/DevOps positions
